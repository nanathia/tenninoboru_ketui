GMSaveBox Class

Super Class

  • GMObject

Overview

ゲームの実行状態を保存し、読み込み直すためのクラスです。

このクラスは、クラスメソッドだけを使って利用します。

Save

void Save()

Discussion

値を設定したあと、この関数を呼び出すまでは、確実に設定された値が保存されることは保証されません。また、この関数が呼び出されなくても、設定された値が保存されることはあります。
